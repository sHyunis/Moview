# 메인홈 주요기능

    - css 디자인
    - css keyframe 을 활용한 타이핑 글자
    - 버튼 클릭 시 메인검색 페이지로 스크롤 다운

# 메인검색페이지 주요기능

    1. fetch를 활용한 Dom연결
    2. 검색창
        - 빈 칸 검색 시 alert메세지
        - 원하는 영화 검색시 해당 영화 카드만 조회
    3. 카드
        - 클릭시 id조회
        - flex를 이용한 반응형 페이지
    4. 고정메뉴바
        - position-sticky를 활용하여 위치고정
        - 이전 버튼 클릭 시 모든 카드 돌아오기
        - 검색 버튼 클릭 시 검색 메뉴로 스크롤
        - 홈 버튼 클릭 시 메인홈으로 스크롤 업
